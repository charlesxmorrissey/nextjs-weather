```bash
# Clone the repo
git clone https://github.com/charlesxmorrissey/nextjs-weather.git

# Copy `.env.example` to `.env` and update environment variables:
cp .env.example .env

# Install dependencies
yarn

# Serve with hot reload at localhost:3000
yarn dev

# Build for production with minification
yarn build

# Serves the site with the Next.js production server
yarn start
```
